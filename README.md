# vuepress-theme-medium

> Medium-Like Vuepress blog theme


## Install

```bash
yarn add vuepress-theme-medium -D
# OR npm install vuepress-theme-medium -D
```

## Usage

```js
// .vuepress/config.js
module.exports = {
  theme: 'medium',
  themeConfig: {
    // Please head documentation to see the available options.
  }
}
```
